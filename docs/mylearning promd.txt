
MyLearning Pro – Specification Document (MVP v1.0)
A streamlined training management portal for MPC workforce.

1. Overview
MyLearning Pro is a modern internal training management platform for MPC workforce (300+ employees) focused on simplicity, Apple-like UX, and efficient administrative workflows.
It enables:
* Tracking training hours
* Managing programs & attendance
* Collecting evaluations
* Generating certificates
* Dashboard analytics
* Reporting to stakeholders
There are two main roles:
* Admin
* Employee
Internal login only (email-based authentication via Supabase).

2. Technology Stack (Suggested)
* Frontend: Next.js / React (Vercel / Netlify)
* DB & Auth: Supabase (Postgres + RLS)
* Backend: Supabase Functions or small Node server (optional)
* Email Sender: Supabase Email / Resend / SendGrid
* Storage: Supabase Storage (certificates)
* Deployment: Vercel / Netlify

3. User Roles
3.1 Role: Admin
Has full access.
Admin capabilities:
* Super Dashboard
* Manage Users (CRUD)
* Manage Training Programs (CRUD)
* Assign programs to employees
* Manage attendance (QR scanning or manual)
* Manage evaluations (view responses)
* Manage certificates (upload, generate)
* Send reminders (auto + manual)
* Export reports (CSV/PDF)
3.2 Role: Employee
Can only view their data.
Employee capabilities:
* View assigned programs
* Submit evaluations
* View evaluation reminders
* Download certificates
* View training history
* View total training hours

5. Features Breakdown
5.1 ADMIN FEATURES
5.1.1 Super Dashboard
Stats shown:
* Total programs this year
* Total participants
* Training hours by department
* Upcoming programs (next 7 days)
* Overdue evaluations
* Compliance % (hours completed vs. target)
* Top 10 and Bottom 10 training hours
5.1.2 Program Management
Admin can:
* Create program
* Edit program
* Delete program
* View list of programs (filter by date, category)
* Assign employees (single or bulk)
* Categories:
o Technical
o Leadership
o Soft Skill
o Mandatory
o Others
Program fields:
title
description
category
start_date_time
end_date_time
location / online_link
organizer
trainer
hours
status (planned/ongoing/completed/cancelled)
5.1.3 Attendance Management
Options:
* QR code check-in
* Manual marking
* Bulk upload (CSV)
Attendance status:
* Assigned
* Registered
* Attended
* No-Show
5.1.4 Evaluation Management
Admin abilities:
* Create evaluation template
* Link to program
* View responses
* Export results
* Trigger reminders
Question types:
* Likert scale
* Radio
* Checkbox
* Open-ended text
5.1.5 Certificate Management
* Upload certificate (PDF)
* Auto-generate certificate (template + dynamic fields)
* Attach certificate to participant(s)
* Allow user downloads
Certificate fields:
name
program
date
hours
signature
organization logo
qr verification (optional)
5.1.6 Reports Module
Admins can export:
* Training hours by user
* Training hours by department
* Participation list
* Program list
* Evaluation summary
* Mandatory program completion
Formats:
* CSV
* PDF

5.2 EMPLOYEE FEATURES
5.2.1 My Trainings
Employee sees:
* Assigned programs
* Upcoming programs
* Completed programs
* Filters (date, category, status)
* Program details page
5.2.2 My Evaluations
* Shows pending evaluations
* Submit evaluations
* Shows completed evaluation history
5.2.3 My Certificates
* List of completed programs
* Download certificate (PDF)
5.2.4 My Training Hours
* Total hours completed this year
* Total lifetime hours
* Hours by category
* Progress bar vs target
